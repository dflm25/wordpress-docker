## Installation

Create .env file

```sh
# Containers NAME Company
APP_NAME=
WORKSPACE_TIMEZONE=America/Bogota
PMA_PPORT=
# Mysql
MARIADB_DATABASE=
MARIADB_USER=
MARIADB_PASSWORD=
MARIADB_ROOT_PASSWORD=
# DB Storage accesible in Principal HOST
DATA_PATH_HOST=./data
```
