## Installation

Create .env file

```sh
# Containers NAME Company
APP_NAME=
WORKSPACE_TIMEZONE=America/Bogota
PMA_PPORT=8082
# Mysql
MARIADB_DATABASE=sites
MARIADB_USER=root
MARIADB_PASSWORD=Lae01.
MARIADB_ROOT_PASSWORD=Lae01.
# DB Storage accesible in Principal HOST
DATA_PATH_HOST=./data
```
